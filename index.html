A solo training tool set for improving Netrunner. Originally built for District Championship prep.

What is Android: Netrunner?

Android: Netrunner is an asymmetric two-player card game set in a cyberpunk future. One player is the Corp, building remote servers, defending them with ice (think firewalls), and trying to score agendas. The other is the Runner, hacking into those servers to steal the agendas. Every decision is a resource calculation: clicks, credits, and cards are finite.

The game is now published by Null Signal Games, a non-profit fan organisation that has continued developing new cards and running organised play since Fantasy Flight Games discontinued the game in 2018.

How to use this trainer

Start by entering your decklists in My decks. The Flashcards tab can draw from those lists to drill your specific cards.

My decks

Paste your Corp and Runner decklists here. Supports standard NetrunnerDB format — one card per line with 3x Card Name or just Card Name. Save each deck separately. The Flashcards tab can use these lists as its card pool when you select "My decks" mode.

Flashcards

Flashcard-style card recognition drill with no pressure. Each card is shown face-up with its artwork. Think about what it does, then hit Reveal to see the oracle text. Rate yourself — Knew it, Unsure, or Blank. Cards you struggle with come up again in a second pass. Use the pool controls to drill just your decks, a specific cycle, or the entire card pool. Your ratings persist between sessions.

Challenge

A levelled progression through the Standard card pool. Six levels follow the sets in order: System Gateway → Elevation → Vantage Point → Ashes Cycle → Borealis Cycle → Liberation Cycle. Each level offers two modes — Set focus drills only that level's new cards; Cumulative drills all cards up to and including that level. Pass a level by correctly identifying 80% of its cards. Levels unlock in sequence and your progress is saved locally.

Glossary

A glossary of the rules and strategic concepts most important for tournament preparation, co-created with Claude AI. Entries cover Mechanics (precise rules definitions), Concepts (established community strategy vocabulary), and Factions (playstyle overviews of all seven main factions). All mechanics entries have been cross-referenced against the current NSG Comprehensive Rules. Use the filter buttons or search to find terms, and follow See also links to explore related entries.

Play helper

An interactive guide to how turns and runs work in Netrunner. Turn structure shows the Corp and Runner turn phases side by side. Run structure is an interactive flowchart — tap YES or NO at each decision point to follow the path of a run in real time. Useful for new players learning why a run plays out the way it does.

The Corp's Turn
1. Draw Phase
  1. Corp gains allotted clicks.
  2. Paid ability window.
  3. Recurring credits refill.
  4. Turn begins.
  5. Corp draws 1 card.
2. Action Phase
  1. Paid ability window.
  2. Does Corp have unspent clicks? NO → go to (d) YES → Corp takes an action
  3. Return to (a).
  4. Action phase ends.
3. Discard Phase
  1. Corp discards cards.
  2. Paid ability window.
  3. Corp loses unspent click.
  4. Turn ends.
  5. Turn is complete. Move to Runner's turn.
The Runner's Turn
1. Action Phase
  1. Runner gains allotted clicks.
  2. Paid ability window.
  3. Recurring credits refill.
  4. Turn begins.
  5. Paid ability window.
  6. Does Runner have unspent clicks? NO → go to (h) YES → Runner takes an action
  7. Return to (e).
  8. Action phase ends.
2. Discard Phase
  1. Runner discards cards.
  2. Paid ability window.
  3. Runner loses unspent click.
  4. Turn ends.
  5. Turn is complete. Move to Corp's turn.

Remember: checkpoints occur at the end of each instruction.

Breaching a Server
  1. The breach begins.
  2. If Archives, turn Archives faceup.
  3. If HQ or R&D, determine number to access.
  4. Any remaining candidates to access? YES → Runner chooses one NO → go to (7)
  5. Runner accesses the chosen card.
  6. Return to (4).
  7. Breaching the server is complete.
Accessing a Card
  1. The card is accessed.
  2. Runner may trash card or use mid-access ability.
  3. If the card is an agenda, Runner steals it.
  4. Access is complete.

Startup
Beginner-friendly

A limited card pool format designed for newer players entering organised play, and for experienced players who want a tighter deckbuilding challenge. Smaller pool means faster games and cleaner decisions.

Card pool as of Vantage Point, March 2026
  • System Gateway + Elevation (Core Sets)
  • Vantage Point

The Liberation Cycle rotated out when Vantage Point was released.

Additional restriction

Startup Corp decks may contain a maximum of 3 agenda cards with a printed agenda point value of 3 or greater.

Standard
Flagship format

The main format for organised play. Most official events, including GNKs and District Championships, use Standard. If a tournament doesn't specify a format, assume Standard.

Card pool as of Vantage Point, March 2026
  • System Gateway + Elevation (Core Sets — never rotate)
  • Ashes Cycle: Downfall, Uprising
  • Borealis Cycle: Midnight Sun, Parhelion
  • Liberation Cycle: The Automata Initiative, Rebellion Without Rehearsal
  • Vantage Point
How rotation works

When new sets are released, older sets eventually leave Standard to keep the card pool fresh. The Core Sets (System Gateway + Elevation) never rotate. Vantage Point's release did not trigger a rotation.

Eternal
No rotation

Eternal uses almost the entire printed card pool across all eras of Netrunner — FFG-era sets included. No rotation ever occurs. It uses a points system instead of a simple ban list: each deck has 7 points to spend on powerful cards. Any card not on the points list costs 0 and can be included freely.

How the points list works

Cards are rated 1–4 points based on power level. Spending a card's point cost lets you include a full playset. A small number of cards are banned outright. You can mix and match across point values as long as your total stays at 7 or under — per deck, not shared between Corp and Runner.

Other formats

Random Access Memories (RAM) uses a randomly drawn card pool, generated in a livestreamed event. Regular RAM tournaments are currently on hiatus but unofficial play continues.

Snapshot is a preserved snapshot of the meta from Magnum Opus 2018, the final FFG organised play event. A piece of Netrunner history — rarely played competitively today.

The community also runs many unofficial formats. Check the NSG Discord and community hubs for what's active near you.
